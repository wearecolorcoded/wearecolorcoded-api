# Install Dependencies
`yarn install`

# Create .env file in root of project
Specify Values for these variables
```
NODE_ENV=<node env here>
NODE_PORT=<node port here>

MONGO_DB_HOST=<mongo db host here>
MONGO_DB_NAME=<mongo db name here>
```

# For Development
`npm run start-dev`

# For Production
`npm start`
